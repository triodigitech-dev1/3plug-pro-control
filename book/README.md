# 3plug Control Book

This folder is the onboarding book for the current Triotek/3plug version of Press.

We will use it to document the real workflow of the product as it exists now:

* what 3plug Control is supposed to do
* how an operator should use it
* the first end-to-end walkthroughs
* the current gaps, assumptions, and lessons from live setup

This `README.md` is the first page of the book.

## Why this book exists

The repo now has a working production setup path, but we still need a clear operator-facing explanation of the actual workflow inside the control panel.

This book will become that explanation.

## Initial focus

The first onboarding flow we want to explain is:

1. understand what this Triotek-maintained Press version is for
2. understand the main operator workflow
3. register the first managed server
4. onboard the first real bench
5. import and inspect the managed sites
6. follow jobs, plays, and forensic signals
